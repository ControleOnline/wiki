# ControleOnline Wiki

Este repositório público espelha o conteúdo público de `ajuda.controleonline.com`.

## Regras de segurança

- Não versionar segredos, credenciais, backups, logs ou arquivos de ambiente.
- O deploy para o FTP usa apenas GitHub Secrets.
- A sincronização do conteúdo deve filtrar arquivos sensíveis antes de qualquer commit.
